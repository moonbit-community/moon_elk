# Milky2018/moon_elk

A MoonBit port of Eclipse Layout Kernel (ELK).

- Repository: https://github.com/moonbit-community/moon_elk.git
- Keywords: elk, layout

## Source Layout

- `plugins/`: layered source tree aligned with `elk-reference` plugin packages.
- `test/`: layered test tree aligned with `elk-reference` test packages.
- `flat/`: compatibility source root used by MoonBit build (`moon.mod.json` uses `"source": "flat"`).
